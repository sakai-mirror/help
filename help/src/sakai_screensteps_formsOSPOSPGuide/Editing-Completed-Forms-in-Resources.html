Editing Completed Forms in Resources

To edit a form that you have already completed, follow these steps:

  1. Open the Resources tool and the folder containing the form.
  2. Select Edit Content from the Actions pull-down list for the form you want to edit. Sakai displays the completed form.
  3. Make the appropriate changes.
  4. To complete the edit, click Save Changes. Or, to exit without editing the form, click Cancel.
