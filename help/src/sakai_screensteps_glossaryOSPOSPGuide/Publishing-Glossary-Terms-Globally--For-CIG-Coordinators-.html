Publishing Glossary Terms Globally (For CIG Coordinators)

If you are a system administrator, you may publish a glossary term globally (that is, for all sites in a Sakai instance). To do so, log in and follow these steps:

  1. Access the Portfolio Admin site.
  2. Select the Glossary tool. Sakai displays the Glossary tool home page.
  3. Add the term as instructed above. Because you are working in the Portfolio Admin site, Sakai makes this term available to all sites.
