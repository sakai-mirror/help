Problemas ao colar texto do FCKeditor para o Firefox

Devido a certas configurações de segurança no Firefox, você pode ter problemas ao colar texto no FCKeditor quando você tenta clicar com o botão direito e seleciona Colar. As versões anteriores do Firefox não pode permite que você cole usando Ctrl-v também. Para resolver esses problemas, escolha uma das soluções abaixo:

  • No menu Editar, selecione Colar.

  • Pressione Ctrl-v. Se isso não funcionar, atualize para a última versão do Firefox.

  • Alterar as configurações de segurança de modo que o Firefox permita colar com o botão direito do mouse. Para fazer isso no Windows:

    1. Feche todas as janela do Firefox.

    2. Navegue para o diretorios abaixo, onde username é o seu username e ProfileName é o seu nome completo ou uma seqüência de caracteres seguido por ".default":

      Windows 7 ou Vista:

      C:\Users\username\AppData\Local\Mozilla\Firefox\Profiles\ProfileName

      XP:

      C:\Documents and Settings\username\Application Data\Mozilla\Firefox\Profiles\ProfileName
    3. Use um editor de texto, crie um arquivo nomeado user.js no directório.

      Dica: Tenha certeza que seu editor de texto não adiciona uma extensão .txt no final do nome do arquivo. O arquivo precisa ter .js como extensão.

    4. No arquivo, digite: user_pref("capability.policy.policynames", "allowclipboard"); user_pref("capability.policy.allowclipboard.sites", "https://www.samplesite.com"); user_pref("capability.policy.allowclipboard.Clipboard.cutcopy", "allAccess"); user_pref("capability.policy.allowclipboard.Clipboard.paste", "allAccess");

      Substitua https://www.samplesite.com com o nome do site que você deseja permitir o colar. Não use / (barra).

    5. Reinicie o Firefox.
