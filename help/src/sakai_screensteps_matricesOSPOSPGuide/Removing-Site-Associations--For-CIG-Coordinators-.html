Removing Site Associations (For CIG Coordinators)

When a site association is removed, all linked submissions (i.e., linked assignments or matrix cells) can no longer be accessed via the matrices in the current site.


To remove site associations:

  1. From the Tool Menu of the portfolio site, choose Matrices.
  2. Choose Manage Site Associations.
  3. Select the sites by checking the boxes, and click Remove Association.
  4. Click Remove, when prompted, to confirm.
