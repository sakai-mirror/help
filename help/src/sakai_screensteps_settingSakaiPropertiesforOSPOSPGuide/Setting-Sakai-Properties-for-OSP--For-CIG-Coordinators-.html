Setting Sakai Properties for OSP (For CIG Coordinators)

Certain Sakai properties are relevant for the Open Source Portfolio (OSP) tools. These properties should be set by a system administrator.

For the full reference of all Sakai properties, consult:

http://confluence.sakaiproject.org/display/DOC/Sakai+Properties+Reference

For current OSP properties, consult:

http://confluence.sakaiproject.org/display/DOC/Sakai+Properties+Reference+-+OSP
